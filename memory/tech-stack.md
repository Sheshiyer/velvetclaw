# Technology Stack

## Core Platform
- **Agent Framework**: OpenClaw (autonomous AI agent framework)
- **Memory**: Markdown files + SQLite semantic search
- **Coordination**: ClawVault primitives (markdown + YAML frontmatter)
- **Skill Registry**: ClawHub (community skill marketplace)

## Models
- **Primary**: GPT-5.3 Codex (OpenAI) — code generation, complex reasoning
- **Fallback**: Gemini 2.5 Flash (Google) — fast inference, cost-effective
- **Thinking Mode**: High — for strategic and analytical tasks

## Infrastructure
- **Dashboard**: Next.js 15 + Convex (real-time database)
- **Hosting**: Vercel (dashboard) / Local (agents)
- **Version Control**: GitHub
- **CI/CD**: GitHub Actions

## Communication Channels
- **Telegram**: Primary user-agent interface
- **Discord**: Community and team coordination
- **GitHub**: Code review and development workflow

## Key Integrations
- **Camofox Browser**: Anti-detection web browsing for remote agents
- **Obsidian**: Human-readable vault UI for agent workspaces
- **Apple ecosystem**: Notes, Reminders, Things (macOS-native productivity)
